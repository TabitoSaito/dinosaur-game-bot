# dinosaur game bot
Simple bot that plays the chrome dinosaur game. Learn project to better understand image processing and automation.

## Features
- detects all obstacles and evades them

## Tech Stack
Python, PIL, pyautogui, cv2

## Installation
```bash
pip install -r requirements.txt
python main.py
```

- navigat to [the dinosaur game](https://elgoog.im/dinosaur-game/)
- press "1" to start the bot
- press "q" to deactivat the bot
